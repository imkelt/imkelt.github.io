    Qt5.14安装OpenCV4.4.0

    1.概述

    本次的主角是OpenCV，OpenCV是一个开源的计算机视觉程序库，为了能够在Qt上调用OpenCV，进行计算机视觉应用程序开发，我们首先得编译OpenCV源代码，以得到适合Qt当前所用编译器的库文件。

    本文将介绍如何使用MinGW对OpenCV源代码进行编译，适用于那些使用Qt，并用MinGW编译器的朋友。

    本次编译过程的环境，win10+Qt5.14+MinGW730+OpenCV4.4.0+cmake3.18

    2.安装前准备

    首先，你需要准备的东西有Qt、OpenCV源代码和cmake：

    （1）下载并安装Qt14.2

    下载地址：http://download.qt.io/archive/qt/5.14/5.14.2/

    文件名：qt-opensource-windows-x86-5.14.2.exe

    安装的时候选择MinGW编译器：

    安装完成后，将Qt安装目录下Qt5.14.2/Tools/mingw730_64/bin添加到Path中，如我的是这样：D:\Qt\Qt5.14.2\Tools\mingw730_64\bin

    （2）下载并安装cmake

    下载地址： https://cmake.org/download/

    文件名：cmake-3.18.1-win64-x64.msi

    安装过程中有个选项是把路径添加到所有用户的Path的，要勾选上，还有一个是生成桌面icon的，也选上。

    （3）下载opencv

    下载地址：https://opencv.org/releases/

    文件名：opencv-4.4.0-vc14_vc15.exe

    下载后双击进行源文件的提取，可以放置某个文件夹里，最好命名下OpenCV-4.4.0，里面有一个source文件夹和一个build文件夹，如果是用VS2015，那么build/x64/vc14中的库就可以直接用，不需要再去编译，如果用的是MinGW，则用不了，需要重新编译。

    3.编译opencv

    （1）第一次Configure

    build文件可以自己建一个，最后用来存放编译结果的，记住这个build文件

    上面的填好后，左下边点击Configure，然后选择使用MinGW进行编译：

    选择C和C++编译器，这两个编译器在前面安装Qt后添加到Path环境变量里的bin文件夹里，如我的是：D:\Qt\Qt5.14.2\Tools\mingw730_64\bin\

    （2）第二次Configure

    点击Finish后会执行第一次配置，第一次Configure完成后进行一些相关的设置：

    不一一截图了，列出在这里：

    • 不勾选 WITH OPENCL D3D11_NV

    • 不勾选 WITH_IPP

    • 不勾选 ENABLE_PRECOMPILED_Header

    • 不勾选 OPENCV_ENABLE_ALLOCATOR_STATS

    • 勾选 WITH_OPENGL

    • 勾选 WITH_OPENMP

    • 勾选 WITH_QT

    • 勾选 BUILD_opencv_world

    第二次点击Configure：然后修改一下Qt5_DIR的Value：

    如我的：D:\Qt\Qt5.14.2\5.14.2\mingw73_64\lib\cmake\Qt5

    （3）Generate

    第二次Configure完成后，左下角点击Generate:

    （4）mingw32-make -j (线程数量)

    打开PowerShell，进入刚刚的build文件，使用命令：

    mingw32-make.exe -j (线程数量)

    （线程数量根据自己电脑cpu性能酌情填写。4核CPU就填4，8核就填8，以此类推，例如我是4核心的就填“mingw32-make.exe -j 4”即可）

    make完成后，输入一下命令进行安装：

    （5）mingw32-make.exe install

    mingw32-make.exe install

    这步比较快，install完成后，整个编译工作就完成了：

    （6）install文件夹

    可以看到build文件中有一个install文件夹：

    进入install文件夹：

    • include里面是头文件

    • x64/mingw/lib/libopencv_world440.dll.a就是编译出来的库文件

    • x64/mingw/bin里的是一些可执行程序，将其添加的Path环境变量中

    4.Qt调用OpenCV库

    新建一个Qt工程，右键点击工程，选择添加库：

    选择添加外部库：

    库文件：install/x64/mingw/lib/libopencv_world440.dll.a

    包含路径：install/include

    在main.cpp文件中添加头文件和一些代码测试OpenCV是否安装完成

    选择release构建方式，由于MinGW默认使用release方式进行编译，所以上边编译出来的是release版本的库，只能用于release方式

    安装完成！！！
